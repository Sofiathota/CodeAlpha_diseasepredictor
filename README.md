# Multi Disease Prediction System Using Machine Learning

## Overview
This project predicts multiple diseases using machine learning techniques and medical data.

## Diseases Included
- Diabetes
- Heart Disease
- Breast Cancer
- No Disease

## Technologies Used
- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy

## Features
- Real-time prediction
- Interactive UI
- Machine learning classification
- Multi-disease prediction system

## Run Project

```bash
streamlit run app.py
